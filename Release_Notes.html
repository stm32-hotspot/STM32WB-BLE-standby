Purpose

This package contains a specific application to demonstrate low power standby feature on STM32WB55 MCU, see README.md file for more details.

This example is based on the STM32CubeWB_v1.14.0 package. For further information about these components, see official package.

STM32WB-BLE-standby example is being delivered as-is, not necessarily supported by ST.

Update History

Main Changes

Initial release of BLE_p2pServer with standby application on STM32WB55 MCU

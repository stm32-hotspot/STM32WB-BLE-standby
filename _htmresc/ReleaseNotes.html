STM32CubeWB Firmware Package V1.14.0 / 08-June-2022

Main Changes

Maintenance Release for BLE, 802.15.4, Thread and Zigbee updates:

  • BLE updates:
    • Support BLEv5.3
    • ThreadX OS support
  • Zigbee updates:
    • BDB 3.0.1 & ZCL 8 update
    • Network Leave management update
    • TX power setting of Zigbee before establising network
    • Manufacturer ID persistence on ZbStartup
    • Multichannel Scan fix
    • Export ZbNlmeEdScanReq on M4 side
    • ThreadX OS support with two new applications (Zigbee_OnOff_ClientRouter_ThreadX)
    • New Zigbee applications (Zigbee_OnOff_ChannelsAgility)
  • Thread updates:
    • Support Threadv1.1 based on Openthreadv1.2 (commit ec02abc47c6d7663f82132d64555839a5c93e018)
      • N.B.: This revision is not certified do not pass 5.2.6 & 5.2.7 GRL test.
      • Certifiable solution can be done using FW 1.13.x
    • Provide the choice to customer to handle Thread NVM at application processor level
    • ThreadX OS support​​​​​ (hread_Coap_Generic_ThreadX)

    • Maintenance Release for CMSIS, HAL/LL Drivers and Projects updates:

  • HAL/LL Drivers updates:
    • Update HAL/LL drivers to include latest corrections
      • Update of HAL COMP, EXTI, FLASH, GPIO, I2C, IRDA, LPTIM, PKA, PWR, RCC, SAI, SMARTCARD, TIM, UART, USART and USB drivers
      • HAL code quality enhancement for MISRA-C Rule-8.13 by adding Cont qualifiers
      • Update of LL COMP, I2C, LPTIM, RCC, TIM, UART, USART and USB drivers
    • Refer to release notes for further details
  • HAL/LL Drivers updates:
    • Update of FW projects following changes in latest version of HAL/LL and Middlewares
